Comment avons-nous procédé?

 Photographie :
Plusieurs photographies prises sur la promenade des anglais pour notre projet. Boitier - Canon 700D avec des objectifs 50mm et 18-55mm.


 Coding :
Création du site web en tant que support, entièrement développé en HTML/CSS & Javascript (Sans aucun CMS).


 Webdesign :
Création d’une maquette sous Photoshop ansi que d’un logo pour le projet. Réalisation de chaque page ainsi que du style guide pour le dévelopement. Création d’icônes personnalisées.


 Photo-manipulation sous Photoshop :
Utilisation de photographies prises par notre groupe, détourage, colorimétrie avancée, ajouts d’effets statiques sur les images.


 Images libres de droits :
Utilisation d’images libres de droits pour la composition futuristes des images originales de notre groupe.
